¿Qué aprendimos en el semestre?

Durante el semestre de Introducción a los Sistemas de Computo, exploramos una amplia gama de temas esenciales que sientan las bases para comprender el mundo de la informática. A lo largo del curso, nos sumergimos en los conceptos clave relacionados con el hardware y el software, los sistemas numéricos, los sistemas operativos, las bases de datos, las redes informáticas, el diseño web HTML y los servidores web.

Hardware/Software

Comenzamos nuestro recorrido estudiando el hardware y el software. El hardware se refiere a los componentes físicos de una computadora, como la unidad central de procesamiento (CPU), la memoria, los dispositivos de almacenamiento y los periféricos. Por otro lado, el software abarca los programas y aplicaciones que se ejecutan en la computadora. Aprendimos cómo estos dos elementos trabajan en conjunto para permitir el funcionamiento de un sistema de computo.

Sistemas Numéricos

Nos adentramos en los sistemas numéricos, donde exploramos diferentes bases numéricas como el sistema binario, el sistema octal, el sistema decimal y el sistema hexadecimal. Estos sistemas son fundamentales para comprender cómo se almacenan y procesan los datos en una computadora.

  • Binario: El sistema binario utiliza 0 y 1 para representar valores numéricos
  • Octal: El sistema octal utiliza ocho dígitos para representar valores numéricos
  • Decimal: El sistema decimal utiliza diez dígitos para representar valores numéricos
  • Hexadecimal: El sistema hexadecimal utiliza dieciséis dígitos para representar valores numéricos

Sistemas Operativos

Estudiamos los conceptos básicos de los sistemas operativos, su función principal de gestionar los recursos de la computadora y proporcionar una interfaz entre el hardware y el software. Exploramos distintos tipos de sistemas operativos, como Windows, macOS y Linux, y analizamos sus características y funcionalidades.

  • Windows: Windows es un sistema operativo popular para PC
  • MacOS: macOS es el sistema operativo utilizado en las computadoras Mac de Apple
  • LINUX: El mejor sistema operativo, no leo llorones que opinen diferente
  • Otros OS: Los demas sistemas no se abordaron a pronfundidad, pero podemos mencionar IOS, Android, Unix, ChromeOS

Bases de Datos

Continuamos con las bases de datos, donde aprendimos sobre la organización y gestión de grandes volúmenes de información. Exploramos los conceptos de tablas, registros y consultas en bases de datos relacionales, y también nos familiarizamos con el lenguaje SQL (Structured Query Language) utilizado para interactuar con las bases de datos.

Redes Informáticas

Las redes informáticas fueron otro tema importante que abordamos. Estudiamos cómo los dispositivos se conectan entre sí para compartir recursos y comunicarse a través de redes locales y redes globales como Internet. Aprendimos sobre los protocolos de red, como TCP/IP, y los conceptos de direcciones IP, enrutamiento y seguridad en redes.

  • Red de área personal (PAN) Una PAN permite la conexión de dispositivos personales en un área cercana
  • Red de área local (LAN) Una LAN conecta computadoras en un área local, como una oficina
  • Red de área local inalámbrica (WLAN) Una WLAN permite la conexión inalámbrica de dispositivos en un área local
  • Red de área del campus (CAN) Una CAN conecta diferentes edificios dentro de un campus o complejo empresarial
  • Red de área metropolitana (MAN) Una MAN conecta redes locales en una ciudad o región metropolitana
  • Red de área amplia (WAN) Una WAN conecta redes locales y metropolitanas a gran escala
  • Red privada virtual (VPN) Una VPN permite la conexión segura entre dispositivos a través de una red pública

Diseño Web-HTML

Nos adentramos en el mundo del diseño web utilizando el lenguaje de marcado HTML. Aprendimos cómo crear estructuras y contenido web, aplicar estilos y formatos básicos, e incorporar elementos multimedia en nuestras páginas web.

  • HTML: HTML es el lenguaje de marcado para estructurar contenido en la web
  • CSS: CSS se utiliza para dar estilo a los elementos HTML en una página web
  • JavaScript: JavaScript se utiliza para programar comportamientos y acciones en las páginas web

Servidores Web

Finalmente, exploramos los servidores web y su papel en la entrega de páginas web a los usuarios. Estudiamos los conceptos de alojamiento web, nombres de dominio, servicios de correo electrónico y seguridad en servidores web.

  • Xampp: XAMPP es un paquete de software que ofrece un entorno de desarrollo local para sitios web
  • Liveserver: LiveServer es una extensión de VS Code que permite crear un servidor web local para desarrollo web
  • Portable WebServer: Portable WebServer es una aplicación que permite ejecutar un servidor web portátil desde una memoria USB sin necesidad de instalación
